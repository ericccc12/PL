Criadores da Programação Linear

A programação linear é um método matemático para otimizar o desempenho de sistemas com recursos limitados. Ela lida com a maximização ou minimização de uma função linear, sujeita a um conjunto de restrições lineares. Utilizada extensivamente em economia, engenharia e ciência da computação, a programação linear foi desenvolvida para resolver problemas complexos de alocação de recursos e planejamento operacional.

George Dantzig

George Dantzig foi um matemático americano que é frequentemente considerado o pai da programação linear. Ele desenvolveu o algoritmo Simplex, um dos mais importantes métodos para resolver problemas de programação linear.

Leonid Kantorovich

Leonid Kantorovich foi um economista e matemático soviético que também contribuiu significativamente para o desenvolvimento da programação linear. Ele aplicou os métodos de programação linear na economia, especialmente em problemas de alocação de recursos.

Tjalling Koopmans

Tjalling Koopmans foi um economista americano-holandês que, junto com Dantzig, recebeu o Prêmio Nobel de Economia em 1975 por seu trabalho sobre a teoria das alocações, que inclui contribuições importantes para a programação linear.

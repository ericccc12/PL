Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de fornecer uma introdução abrangente à programação linear e ao Algoritmo Simplex. A programação linear é uma técnica poderosa utilizada em diversas áreas, como economia, engenharia e ciência da computação, para otimizar resultados sob certas restrições.

Essa atividade foi apresentada como atividade complementar na disciplina de Programação Linear do curso de Análise e Desenvolvimento de Sistemas, ministrada pelo professor Luciano Condori, na Fatec de Carapicuíba.

Objetivos

  • Explicar os conceitos básicos da programação linear.
  • Introduzir o Algoritmo Simplex e seu funcionamento.
  • Fornecer recursos visuais e vídeos educativos para facilitar a compreensão.
  • Demais assuntos relevantes.
